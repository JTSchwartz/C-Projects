C Programs

This is a collection of simple programs written in C for a CPS 356 Operating Systems class.

Current Programs:

  • alternate.c
    • This program will fork and synchronize 20 processes so that each process creates a child and they alternate between printing ping and pong.
  • countsubsstdin.c
    • CountSubStdIn will take 2 required inputs, a substring and a full string, as well as a required -nooverlap flag. The program will count the amount of times the substring is found in a string, and the flag will determine if overlap is allowed.
  • removesubsargs.c
    • An adaptation of CountSubsStdIn, this program takes the input as arguments, and returns the count of substrings in the string, as well as the given string with all found substrings removed.

Language: C
